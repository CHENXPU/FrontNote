# 一个轻量版的仿 koa 框架

## 如何运行 demo

由于使用 async/await，因此 node 版本需要高于 7.6 。也可以安装 babel-cli，使用其中的 babel-node 运行。

- 中间件 demo：node examples/middlewareDemo.js
- 错误处理 demo： node examples/extendDemo.js
- ctx 扩展 demo： node examples/extendDemo.js
