# 脏值检查

原理为通过定时器进行轮询查询，但是只在特定事件触发后进行脏值检测

  • DOM 事件，譬如用户输入文本，点击按钮等。( ng-click )
  • XHR 响应事件 ( $http )
  • 浏览器 Location 变更事件 ( $location )
  • Timer 事件( $timeout , $interval )
  • 执行 $digest() 或 $apply()
